How to use Melody Palette?

Step 1

Create a sample melody. You can create sample melody using following ways:

  • Use your own keyboard & audio interface
  • Using this site you can generate your own midi file
    • Link :Online Sequencer

Step 2

Uploading File

  • Click on Choose File button
  • Choose your MIDI Input file
  • Click on Upload button.

Step 3

Downloading File

  • Download button will appear on a screen.
  • Click on download and save the file.
  • You can view the music score using MuseScore Softwere or else you can play the media file on your own system.
    • Get MuseScore : Download MuseScore

About

Melody Palette is a tool to generate melodies using Artificial Intelligence. All the user has to to is to provide a short input. Then this tool will automatically create the melody own its own.
